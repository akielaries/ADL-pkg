84 // As usual we reset the projection transformation whenever the window is
85 // reshaped. This is done (of course) by setting the current matrix mode
86 // to GL_PROJECTION and then setting the matrix. It is easiest to use the
87 // perspective-projection-making matrix from the GL utiltiy library. Here
88 // we set a perspective camera with a 60-degree vertical field of view,
89 // an aspect ratio to perfectly map into the system window, a near clipping
90 // plane distance of 1.0 and a far clipping distance of 40.0. The last
91 // thing done is to reset the current matrix mode to GL_MODELVIEW, as
92 // that is expected in all the calls to display().
93 void reshape(GLint w, GLint h) {
94  glViewport(0, 0, w, h);
95  glMatrixMode(GL_PROJECTION);
96  glLoadIdentity();
97  gluPerspective(60.0, (GLfloat)w/(GLfloat)h, 1.0, 40.0);
98  glMatrixMode(GL_MODELVIEW);
99 }
