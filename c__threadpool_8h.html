◆ dispatch()

void dispatch ( threadpool  from_me,
dispatch_fn  dispatch_to_here,
void *  arg 
)

dispatch sends a thread off to do some work. If all threads in the pool are busy, dispatch will block until a thread becomes free and is dispatched.

Once a thread is dispatched, this function returns immediately.

The dispatched thread calls into the function "dispatch_to_here" with argument "arg".
